# Basic Nodejs app

## Install packages
```bash
yarn install
```

## Create `.env` file and set variable
```bash
APP_NAME="DucXinh App"

DB_HOST=""
DB_USER=""
DB_PASSWORD=""
DB_NAME=""
DB_PORT=""
```

## Create database and table `users` then insert some record to `users` table

## Run
```bash
yarn dev
```

## Result
Open Browser at localhost:8000
